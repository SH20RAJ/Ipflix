IP Details API Documentation

This API provides details about the IP address related to the incoming request.
Endpoint: /api/ipdetails
Method: GET

Response Format

    {
      "timestamp": "2023-07-24T12:34:56.789Z",
      "ip": "123.456.789.0",
      "city": "Sample City",
      "region": "Sample Region",
      "country": "Sample Country",
      "loc": "12.3456,78.9012",
      "org": "Sample Organization",
      "timezone": "UTC",
      "fromYourSide": "This is an additional detail from our side!"
    }
